# Ameblo2WP class

## これは何?

[アメブロ](https://ameblo.jp/)の記事を [WordPress](https://wordpress.org/) にインポートするための PHP クラスです。

## 動作環境

* PHP (5.6.x 以降推奨)
 * exec が実行可能であること
* [WP-CLI](http://wp-cli.org/) (1.2.x 以降推奨)
